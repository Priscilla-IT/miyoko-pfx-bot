## Installation

To launch bot, follow these steps:

```sh
$ git clone https://github.com/Priscilla-IT/miyoko-pfx-bot.git
$ cd miyoko-pfx-bot
$ sudo chmod +x bot
$ ./bot
```

Edit the .env file.
